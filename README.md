# Personal Academic Website

This repository contains the source code for my personal academic website, developed to present my academic journey, teaching portfolio, leadership roles, and professional contact information.

The website serves as a long-term academic identity for interviews, faculty recruitment, and institutional accreditation processes such as NAAC and NBA.

---

## About the Website

The website provides a structured and verifiable overview of my academic profile, including:

- Academic appointments and professional journey  
- Educational qualifications  
- Teaching portfolio aligned with Outcome-Based Education (OBE)  
- Academic leadership and institutional responsibilities  
- Research interests and academic direction  
- Professional contact with direct email and secure message form  

All content is organized carefully to maintain academic integrity and accuracy.
